## 3. Logo Concepts

### Logo Philosophy

The OneDettyDecember logo should be:
- **Instantly recognizable** - Even at tiny sizes (social media avatars)
- **Culturally resonant** - Feels authentically African
- **Modern & Fresh** - Appeals to young, digital-savvy audience
- **Versatile** - Works on light/dark backgrounds, merchandise, etc.
- **Energetic** - Conveys movement, excitement, celebration

### Logo Concept Directions

#### Concept A: "The Pulse"
**Idea:** Abstract waveform/pulse that represents:
- Afrobeats rhythm
- Heartbeat of the celebration
- Energy waves
- Movement and travel

**Visual Elements:**
- Stylized waveform/soundwave
- Could incorporate plane, location pin, or party icons within the wave
- Dynamic, flowing lines
- Gradient colors

**Vibe:** Modern, trendy, music-focused

---

#### Concept B: "The Journey Circle"
**Idea:** Circular emblem that represents:
- The journey home (diaspora returning)
- Unity and community
- Continuous celebration
- Global connection

**Visual Elements:**
- Circular badge/seal design
- Airplane trail circling the badge
- West African map silhouette
- Ankara pattern border
- Palm tree or cultural icon in center

**Vibe:** Heritage, journey, unity

---

#### Concept C: "The Vibrant Wordmark"
**Idea:** Bold, custom typography with cultural flair
- "Detty" in bold, energetic lettering
- "December" incorporated creatively
- Possible tagline: "Tours" or "West Africa's Party"

**Visual Elements:**
- Custom lettering with African pattern fills
- Possible integration of iconic symbols (plane, location pin, party hat)
- Vibrant gradient or multi-color approach
- Could have a subtle Ankara pattern texture overlay

**Vibe:** Bold, direct, culturally rich

---

#### Concept D: "The Icon + Wordmark"
**Idea:** Combination mark with:
- Symbolic icon (can stand alone)
- Clean wordmark

**Icon ideas:**
- Stylized "DD" monogram with African pattern integration
- Palm tree + airplane combination
- Location pin shaped like Africa
- Celebration burst/star

**Vibe:** Professional yet fun, versatile

## 6. Typography

### Font Families

**Primary Font: Inter**
- Modern, clean sans-serif
- Excellent readability
- Great for UI and body text
- Weights: 400 (Regular), 600 (SemiBold), 700 (Bold), 900 (Black)

**Display Font: Bebas Neue**
- Bold, impactful
- Use for headlines, hero text
- All caps
- Weight: Bold

**Alternative Display: Montserrat**
- Geometric sans-serif
- Versatile
- Weights: 700 (Bold), 900 (Black)

**Optional Accent: African-inspired Display Font**
- Research: "African Sans" or custom African typography
- Use sparingly for special headers

### Type Scale

**Headings:**
- H1: 48px / 3rem (Bebas Neue Bold)
- H2: 36px / 2.25rem (Bebas Neue Bold)
- H3: 28px / 1.75rem (Inter Bold)
- H4: 24px / 1.5rem (Inter Bold)
- H5: 20px / 1.25rem (Inter SemiBold)

**Body:**
- Body Large: 18px / 1.125rem (Inter Regular)
- Body: 16px / 1rem (Inter Regular)
- Body Small: 14px / 0.875rem (Inter Regular)

**UI:**
- Button Text: 16px / 1rem (Inter SemiBold)
- Input Labels: 14px / 0.875rem (Inter SemiBold)
- Captions: 12px / 0.75rem (Inter Regular)

### Type Usage

**Headlines:**
```
DETTY DECEMBER 2026 IS HERE

```

**Subheadings:**
```

```
From flights to hotels to events to tours‚Äîplan your entire 
Detty December trip in 10 minutes.

## 7. Brand Applications

### Logo Variations

You'll need multiple versions:

1. **Primary Logo** (full color, horizontal)
2. **Stacked Logo** (full color, vertical)
3. **Icon Only** (for app icon, social media)
4. **White Version** (for dark backgrounds)
5. **Black Version** (for light backgrounds)
6. **Single Color** (for merchandise, print)

### Minimum Sizes

- **Full Logo:** Minimum width 120px (digital) / 1 inch (print)
- **Icon Only:** Minimum 32px √ó 32px (digital)

### Clear Space

Maintain clear space around logo equal to the height of the "D" in "Detty"

### Incorrect Usage

‚ùå Don't stretch or distort
‚ùå Don't change colors arbitrarily
‚ùå Don't add effects (drop shadows, outlines) unless approved
‚ùå Don't place on busy backgrounds without contrast
‚ùå Don't rotate or tilt
